# 知乎热点功能使用说明

## 📋 功能简介

通过集成知乎热点API，实现了实时获取知乎热门话题并生成自媒体文章的功能。用户可以通过聊天室中的命令快速获取热点话题，并基于这些话题创作内容。

## 🎯 使用命令

### 1. 获取知乎热点话题
```
/知乎
```
获取当前知乎最热门的5个话题，包含：
- 话题标题
- 热度值
- 创作提示
- 相关标签
- 原文链接

### 2. 基于热点生成文章
```
/知乎文章 [索引号]
```
基于指定索引的热点话题生成完整文章。

**示例：**
- `/知乎文章 1` - 基于第1个热点话题生成文章
- `/知乎文章 3` - 基于第3个热点话题生成文章

### 3. 搜索特定话题
```
/知乎话题 [关键词]
```
搜索包含指定关键词的知乎话题，并基于搜索结果生成文章。

**示例：**
- `/知乎话题 人工智能` - 搜索AI相关话题并生成文章
- `/知乎话题 创业` - 搜索创业相关话题并生成文章

## 🔄 工作流程

1. **获取热点** → 实时调用知乎API获取最新热点话题
2. **智能筛选** → 自动筛选适合自媒体创作的话题
3. **内容生成** → 基于头条服务生成高质量文章
4. **格式优化** → 自动添加标题、标签、创作提示
5. **一键发布** → 可直接用于头条号或其他平台发布
